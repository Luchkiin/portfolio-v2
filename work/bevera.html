Project Overview

BEVERA is a static web page that serves as an online eCommerce page suitable for companies that sell alcoholic beverages online.

BEVERA is a fictional company that was created for a school project I worked on during my first year as an systems science student. The project acts as a good starting template for eCommerce sites by incorporating a modern sleek design by the use of a fresh color palette, easily usable navigation and high-resolution pictures.

Design Goals

Design a responsive web site template for future company's to apply and use, in relation to its fundamental purpose, as an eCommerce site that sells alcoholic beverages online.

For this project, I have strived for 4 design goals. These are:

  • Create a design that is user-friendly and beautiful.
  • Create a web site that targets the appropriate audience with the right information regarding the selling products.
  • Provide the end-user with the right basic functionality that benefits the overall ease of use for the web page.
  • Create a responsive web design, which enables the user to use the website efficiently whether they are on a mobile, tablet or a PC/Laptop
