Project Overview

Geosty is a weather application for iOS and Android devices. The application is built with C#, XAML and Xamarin Forms framework in Microsoft's VS which enables cross-platform programming. The application offers current, hourly, and daily weather forecast based on the end-users current location alongside with humidity and wind details. Furthermore, the weather forecast information is being retrieved from the weather API from OpenWeatherMap.

Geosty makes use of a fresh blue linear gradient color palette alongside with subtle hints of grey and white colors.

Design Goals

Design a weather application that offers the essential weather details alongside with being user-friendly and beautiful.

For this project, I have strived for 3 design goals. These are:

  • Create a weather application that emphasizes the essential weather forecast/condition information
  • Provide relevant basic functionality that benefits the overall ease of use
  • Create a weather application design that works on both iOS and Android devices

Font(s) used: Nunito Sans
