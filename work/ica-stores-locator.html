Project Overview

ICA Stores Locator is a JavaScript map application that enables the user to find different ICA stores located in the Dalarna (SE) province.

By using this application, the user can zoom in on a selected ICA store by clicking on it in the list item menu on the left-hand side. Once zoomed in, the user can then see relevant store information, current weather of that location, and nearby bus stops with its next departing time.

Design Goals

Design a responsive map application that enables the user to zoom in seamlessly on a specific store to find out more information. The project is defined and developed by four design goals. These are:

  • Create a design that is user-friendly and simple.
  • Create an application that targets the appropriate audience with the right information regarding the different ICA stores.
  • Provide the end-user with the right basic functionality regarding map functions so that it benefits the overall ease of use of the application.
  • Provide extra functionalities, such as current weather and near bus stops, to enhance eye-opening user interaction for the end-user.
