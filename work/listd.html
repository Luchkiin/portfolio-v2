Project Overview

Listd is an Android application that serves as a to-do list which lets you save your daily tasks. Listd put emphasis on simplicity and ease of use. Whether you want a shopping list, grocery list or you just have lots of things to remember Listd is built for you.

Listd was initially created as a school project, in which I worked together with my brilliant classmate Najem Seddiqi. This project was done during my final year as a systems science student for a course about the design of mobile systems which covered the development of Android applications.

The application makes use of the Volley HTTP Android library which enables networking easier and faster. Furthermore, the application works against a back-end structure that lets the user perform CRUD operations with the choices that are being executed.

Design Goals

Design a mobile application that users can apply and use to acts as a to-do list by keeping your daily life organized and time-efficient. The project is defined and developed by two design goals. These are:

  • Create a design that is user-friendly and simple.
  • Create an application that offers the essential and most important functions for a to-do list.
