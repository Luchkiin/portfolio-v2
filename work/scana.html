Overview

SCANA is a simple mobile application that offers bar/QR code scanning for both iOS and Android devices. The application is built with C#, XAML and Xamarin Forms. The application offers a modern sleek user interface design that the end-user can integrate with, and a fresh green color palette with subtle hints of grey, black and white colors.

SCANA was initially created as a real-life case collaboration between a known IT company in Borlänge, SE and a group of students (including me) for a course I enrolled in called "Systems Development on the Internet/Intranet". The project was developed during my final year where we got assigned the task of creating an early prototype in the form of a mobile QR code scanning application and an Azure web application. These two applications would in the final prototype version be able to communicate with each other, that consisted of a solid front/back-end structure which acted as a document manager to maintain important product documents for end-users to access and view when required.

However, the shown application is a stripped version of the original application and only highlights the mobile application (i.e. the bar/QR scanner application).

Design Goals

Design a mobile scanning application in which the end-user can use as their go-to bar/QR scanner. The project is defined and developed by three design goals. These are:

  • Create a scanning application that offers the essential functionality of a bar/QR scanner.
  • Provide a sleek and modern user interface design for the end-user to use and integrate with.
  • Put emphasize on the basic functionality which matches that of a universal scanning application.
