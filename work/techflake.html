Project Overview

Techflake is a static web page that serves as a providing online service/support page for companies with different products and services.

Techflake is a fictional company replacement of a project I once worked with at OpusCapita during my summer internship in 2019. This project acts as a good starting template for an online service/support page that incorporates a modern sleek design by the use of a fresh color palette and easily usable navigation.

Design Goals

Design a responsive web site template for future company's to apply and use, in relation to its fundamental purpose, as an online service/support page.

For this project I have strived for 4 design goals. These are:

  • Create a design that is user-friendly and beautiful.
  • Create a web site that targets the appropriate audience.
  • Create a design that features a user-interface that flows openly, i.e. allowing end-users to navigate easily and find what they need quickly and efficiently.
  • Provide the end-user with the right basic functionality that benefits the overall ease of use for the web page.
